1. DOM is a object model for the html document.
2. DOM has 3 parts: 
    Core DOM ( which represent the html elements as nodes in the low level language)
    DOM api ( which we use in our JS code, to perform various document operations : This api is standardised by W3c. )
    CSSOM and LAyouting: attach styles to each HTML DOM node, and calculate the position on screen.
3. querySelector: uses cached value of DOM, getElementsById() : uses latest value of DOM. ( after all children are appended/removed ).
4. Add: append, appendChild, prepend, prependChild, before, after.
