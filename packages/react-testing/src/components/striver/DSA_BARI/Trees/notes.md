1. Binary tree: A tree with max 2 children; min 0 children.
2. Full binary tree: either has 0 or 2 children.
3. Complete BT: all levels are completely filled except the last level. The last level has all nodes on left.
4. Perfect BT: all leaf nodes at same level.
5. Balanced: Max height of tree is (log2n);
6. Degenrate: Skew tree. (same as LL). Every node has single child.
